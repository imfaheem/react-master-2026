What is React?

React is a JavaScript library used to build fast, interactive, and reusable user interfaces.

Why was React created?

React was created to simplify building complex user interfaces by reducing manual DOM manipulation and promote reusable, component-based architecture.

Is React a library or a framework?

React is a JS Library, not a framework.

Difference between Libraray and a Framework.

A Library is collection of pre-defined code that we use to build applications easier and faster. It helps to reduce code duplication and improve productivity.

A Framework provides a pre-defined architecture and controls the overall structure and flow of an application.

How does React differ from vanilla JavaScript when updating the UI?

In React, we update the state, and React automatically re-renders and updates the UI. While in Vanilla JS, we manually manipulate the DOM to update the user interface.

What is Vite?

Vite is a modern frontend build tool used to create, develop, and build web applications quickly.

Why is Vite preferred over CRA?

Because it starts faster, rebuilds faster, and provides a better development experience than CRA (create-react-app).

What is JSX?

JSX is JavaScript XML which is a syntax extension that allows us to write HTML-like code inside JavaScript. It looks like HTML but it's not purely HTML.

Why we use JSX instead of HTML directly?

Because JSX makes React components easier to read and write. JSX is not actual HTML, it is converted into JavaScript by different compilers or transpilers like Babel or esbuild that convert JSX into JS.

What is a Component?

A component is a reusable piece of UI written as a JS function or sometimes also written as a JS class.

Why we use Components?

We use components to make the code reusable, easier to maintain and update. Edit it once, and every page is updated.

Types of Components?

There are mainly 2 types of components.

1. Functional component: The UI code is written inside a function.

2. Class components: The UI is defined inside a class.

What are Props?

Props are the properties that are used to pass data from a parent component to the child component. Internally, props are passed as a read-only object.

Why Do We Need Props?

We use props to pass data from parent component to child component. This allows us to reuse the same component with different data instead of creating duplicate components.

Why we use keys in React?

We use keys for unique identification of each item in the list. It helps React to identify which item is added, deleted, or changed so that it can update only necessary parts of the UI.

What is Conditional Renerding?

Conditional rendering is a technique in React used to display different statements, components or UI based on a condition. It can be implemented using if statements, the ternary operator or the logical AND operator.

When does React re-render?

React re-renders when a component's state change or its props changes or a parent component re-renders.

What are Events in React?

An event is an action or an interaction performed by a user or browser to interact with elements such as Buttons or Input fields. React listens to these events and executes the associated event handler function.

What is an Event Handler?

An event handler is a function that is executed when a specific event occurs. We use event handlers to respond to user interactions and perform actions such as updating states, submitting forms, or calling another function.
